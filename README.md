# 错题整理器 · Wrong Question Organizer

英语语法填空错题整理,按学生错误类型分为 6 类,共 38 题。

## 文件

| 文件 | 用途 |
|---|---|
| [`错题分类整理.html`](错题分类整理.html) | **完整讲解版** —— 每题含错误答案、正确答案、考点,并附错误类型统计诊断,适合复习/打印 |
| [`错题自测版.html`](错题自测版.html) | **交互自测版** —— 答案默认打码,点击每行揭示;支持一键全部显示/隐藏,适合做题自测 |

## 错误类型分布

| 类型 | 题数 |
|---|---|
| 一、词性 / 词形判断 | 9 |
| 二、非谓语动词形式 | 8 |
| 三、谓语(时态 / 语态 / 主谓一致) | 6 |
| 四、从句连接词 | 6 |
| 五、介词 / 固定搭配 | 7 |
| 六、冠词 / 代词 | 2 |

主要薄弱点集中在:**非谓语动词、词性判断、介词搭配**。

## 使用

直接用浏览器打开任一 HTML 文件即可,均为纯离线单文件,无需联网或依赖。
